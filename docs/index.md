## About Akylas

Akylas is a mobile app development consultancy specializing in creating high-quality, performant applications for iOS and Android. With deep expertise in cross-platform development using NativeScript and modern JavaScript frameworks, we help businesses bring their ideas to life.

### What We Do

- **Custom Mobile Applications**: Native-quality apps built with cross-platform technologies
- **Open Source Development**: Contributing to and maintaining popular open-source projects
- **Technical Consulting**: Helping teams make the right technical decisions
- **Code Reviews & Audits**: Ensuring code quality and best practices

### Technologies

**Mobile**: NativeScript, iOS, Android, Titanium Mobile  
**Web**: Vue.js, VitePress, Node.js, TypeScript  
**Tools**: Git, GitHub Actions, Fastlane, VS Code
