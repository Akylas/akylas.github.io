# Activelook

![Activelook](/img/portfolio/activelook/90ViCQSM1BCVUtaYrL2nbnhvdAFN_lHQJ6Pze8K-XEo3SJbTVOB9RpY0aJgTF4QJ2onS=s360.png)

## Overview

**Activelook** is an innovative companion application for smart glasses designed for athletes and outdoor enthusiasts. The app connects to Activelook smart glasses via Bluetooth, providing real-time performance data directly in the user's field of vision.

### Project Details

- **Platform**: iOS & Android
- **Technology**: NativeScript
- **Release Date**: January 2018
- **Category**: Sports & Fitness, IoT

## Features

### Smart Glasses Integration

- 📱 **Bluetooth Connectivity**: Seamless pairing with Activelook smart glasses
- 👓 **Display Control**: Customize what information appears on glasses
- 📊 **Real-time Data**: Live performance metrics during activities
- ⚙️ **Configuration**: Full control over glass display settings
- 🔋 **Battery Management**: Monitor and optimize battery usage

### Performance Tracking

- 🏃 **Activity Monitoring**: Track running, cycling, and other sports
- ⏱️ **Time & Distance**: Accurate time and distance measurements
- 💓 **Heart Rate**: Integration with heart rate monitors
- 📈 **Performance Analytics**: Detailed post-activity analysis

### User Experience

- Clean, intuitive interface
- Easy device pairing and setup
- Customizable data fields
- Activity history and trends
- Social sharing capabilities

## Technical Implementation

**Framework**: NativeScript  
**Connectivity**: Bluetooth Low Energy (BLE)  
**Data Sync**: Real-time data streaming  
**Storage**: Local caching and cloud backup  

### Technical Challenges Solved

- **BLE Communication**: Reliable Bluetooth connectivity in various conditions
- **Real-time Streaming**: Low-latency data transmission to glasses
- **Battery Optimization**: Minimal power consumption during long activities
- **Cross-platform BLE**: Unified BLE implementation for iOS and Android
- **Firmware Updates**: Over-the-air updates for smart glasses

- Athletes and runners
- Cyclists
- Triathletes
- Outdoor sports enthusiasts

## Innovation

Activelook represents the cutting edge of wearable technology, bringing heads-up display functionality to sports and outdoor activities. The mobile app serves as the control center, making advanced smart glasses technology accessible and user-friendly.

## Impact

- Enhanced athlete safety with eyes-up data display
